# Titanic Data Cleaning & Preprocessing

## Objective
The objective of this project is to clean and preprocess the Titanic dataset for data analysis.

## Dataset
Titanic Dataset (Kaggle)

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Tasks Performed
- Handled missing values
- Removed duplicate records
- Renamed columns
- Checked data types
- Prepared the dataset for analysis

## Outcome
A clean and well-structured dataset ready for analysis and machine learning.
